# `data` - Responsabilidade da Pasta

Esta pasta agrupa todos os dados usados na Entrega 2.

## Subpastas

- `raw/`: dados originais sem modificação.
- `external/`: dados de terceiros prontos para entrada.
- `interim/`: dados intermediários após transformações parciais.
- `processed/`: dados finais preparados para análise/publicação.

## Regra geral

Evitar misturar estágios de processamento para garantir rastreabilidade do pipeline.
